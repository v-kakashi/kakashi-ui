# kakashi-ui

> 又一个组件库

## 说明

基于 vue 的一套 ui 组件库，借 (chao) 鉴 (xi)  [mint-ui](http://mint-ui.github.io/#!/zh-cn) , [crossjs](http://crossjs.com/plato/#/) 组件。

## 特点
1. 逻辑代码与[样式](https://github.com/v-kakashi/kakashi-theme)分离
2. 按需加载不同组件。
3. 构建出统一风格的组件。
4. 友好的文档说明和例子。
