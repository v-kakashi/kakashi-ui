属性

参数 说明 类型 默认值
class 图标 className String ''
src 图片路径 String 1 像素图片
alt img 标签的 alt 属性 String ''
title img 标签的 title 属性 String ''
width 图片的宽度 String, Number ''
height 图片的高度 String, Number ''
keepRatio 保持图片的缩放比例 Boolean true
