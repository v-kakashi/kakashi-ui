## vkIcon

| 参数      | 说明                                     | 类型       | 默认值 |
|-----------|------------------------------------------|------------|-------|
| value | 图标的图案 | String  | ''    |
| color | 图标颜色 | String | '' |
| class | 图标的 className | String | '' |
