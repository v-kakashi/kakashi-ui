使用说明

说明

基于 vue 的一套 ui 组件库，借 (chao) 鉴 (xi) mint-ui , crossjs 组件。

特点

  1. 逻辑代码与样式分离
  2. 按需加载不同组件。
  3. 构建出统一风格的组件。
  4. 友好的文档说明和例子。

安装

  npm install kakashi-ui

使用

按需加载

  import vkButton from 'kakashi-ui/src/Button'
  import 'kakashi-theme/src/components/button.less'
